# Connectsy-video-chat-app

Connectsy is a video chat application for direct calls and group calls.

**Technologies Used**

- WebRTC
- JavaScript
- React
- Redux
- Socket.io
- Express.js
- Node.js

**Contributors:** [Muskaan Gupta](https://github.com/muskaan1709)

---

## Features

- Direct Calls
- Group Calls
- Video Streaming
- Text chat
- Mute Video/Audio
- Screen Sharing

## Installation
### Clone
- Clone this repo to your local machine using `https://github.com/muskaan1709/Connectsy-video-chat-app.git`

### Setup
**Client**
> Move to client folder, update and install this package
<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm start
  </code>
</pre>

**Server**
> Move to server folder, update and install this package
<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm run dev
  </code>
</pre>
